# order-system-vue-fastapi

## 概要（What）

飲食店向けの注文管理のプロトタイプ  
音声認識、AUレコメンド機能、Vuetifyによるモダンなデザインの機能検証

## 特徴（Why）

- Vue 3 + Vuetify 
- FastAPI + SQLite + SQLAlchemy
- Whisper | vosk
